# flav-u-crave
it's the bomb.com

## Installation:
```
git clone https://github.com/knowbodynos/flav-u-crave.git
```

## Usage:
```
usage: download_chunk.py [-h] [--out-dir OUT_DIR]
                         [--chunk-length CHUNK_LENGTH]
                         url

Download and chunk audio and text from youtube video.

positional arguments:
  url                   youtube video url

optional arguments:
  -h, --help            show this help message and exit
  --out-dir OUT_DIR, -o OUT_DIR
                        output directory
  --chunk-length CHUNK_LENGTH, -c CHUNK_LENGTH
                        max length of chunks in seconds
```
